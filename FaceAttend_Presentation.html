📝 Module 1

Student Registration

  • 👤 Enter name, roll number & department
  • 📷 Camera captures 5 face samples
  • 🔍 Blur detection rejects low-quality images
  • 🧮 5 descriptors are averaged for accuracy
  • ☁️ Photo uploaded to Cloudinary
  • 🗄️ Data stored in MongoDB Atlas
📸 → 🧠 → 💾

Multi-Sample Capture

5 samples with different micro-angles create a robust face profile that works even with slight head movements

📸 Module 2

Attendance Scanning

🟢

Recognized

Green box + name + confidence % shown on screen. Attendance auto-marked.

🟡

Liveness Check

Orange box = face matched but user must move slightly to confirm they're real (not a photo).

🛡️ Liveness Detection (Anti-Spoofing)

Tracks nose-tip landmark movement across 3+ frames. Static photos have zero movement and get rejected.
